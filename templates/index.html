🌲 Model Info — TF-IDF + Isolation Forest
TF-IDF Vectorizer
Converts preprocessed log tokens into numerical feature vectors.
max_features: 500
n-grams: (1,2) unigrams + bigrams
sublinear_tf: True (log scaling)
Isolation Forest
Unsupervised anomaly detection. Isolates outlier log vectors using random partition trees.
contamination: 10%
estimators: adaptive (n/5)
no labels needed
Anomaly Scoring
Raw scores normalized to 0–1 range. Logs with score < threshold marked as anomalies. Displayed per-row in the processed table.
Lower score = more anomalous
